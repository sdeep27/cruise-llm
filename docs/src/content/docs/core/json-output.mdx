Force the model to return valid JSON.

```python
from spaceshift import LLM

result = LLM().sys("Return JSON with keys: name, age").user("Alice is 30").result_json()
print(result)  # {"name": "Alice", "age": 30}
```

## Outline

- `.chat(json=True)` — chat with JSON mode enabled, keeps history, returns self
- `.result_json()` — run in JSON mode, return parsed dict, reset history
- `.run()` — template interpolation + JSON mode for function-style usage
- JSON schema enforcement via system prompts
- Error handling when JSON parsing fails
- When to use `.chat(json=True)` vs `.run()` vs `.result_json()`
