Manage conversation history and persist LLM state.

```python
from spaceshift import LLM

llm = LLM(model="best").user("Explain dark matter").chat()

llm.save_llm("agents/researcher.json")
loaded = LLM.load_llm("agents/researcher.json")
llm.to_md("conversation.md")
```

## Outline

- `.compact()` — summarize conversation to reduce context length
- `.save_llm(path)` — serialize full LLM state to JSON
- `.load_llm(path)` — restore LLM from saved state
- `.to_md(path)` — export conversation as formatted markdown
