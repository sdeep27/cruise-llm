## Install

```bash
pip install spaceshift
```

## API Keys

Your access to models is based on your API keys from the various providers. Create a `.env` file in your project root:

```bash
OPENAI_API_KEY=sk-proj-...
ANTHROPIC_API_KEY=sk-ant-...
GEMINI_API_KEY=AIza...
TOGETHERAI_API_KEY=...
XAI_API_KEY=xai-...
```

You only need keys for the providers you want to use. Most providers offer free API keys.

## Verify

```python
from spaceshift import LLM

LLM().user("Hello").chat()
```
