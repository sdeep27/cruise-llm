The core research loop: **branch** into questions, **navigate** the space of perspectives, **evaluate** to find what works best.

## Outline

- The three-phase loop: branch ‚Üí navigate ‚Üí evaluate
- Branch: decompose topics with `research_tree`, `subprompt`
- Navigate: move through prompt space with `superprompt`, `sideprompt`, `language_transform`, `prompt_transform`
- Evaluate: rank outputs with `pairwise_evaluate`, `compare_models`, `grid_search`
- How the pieces compose together
- When to use which tool
- Structured markdown output as the common thread
